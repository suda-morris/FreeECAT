enum ec_master_phase_t

EtherCAT master phase.

Enumerator
EC_ORPHANED 

Orphaned phase.

The master has no Ethernet device attached.

EC_IDLE 

Idle phase.

An Ethernet device is attached, but the master is not in use, yet.

EC_OPERATION 

Operation phase.

The master was requested by a realtime application.
